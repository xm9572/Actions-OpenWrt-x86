此版本 openwrt 包含服务：
- passwall
- ShadowSocksR Plus+
- 阿里云盘 WebDAV
- 上网时间控制
- 解锁网易云灰色歌曲
- OpenClash
- 动态 DNS
- 迅雷快鸟
- 网络唤醒
- Frps
- UPnP
- KMS 服务器
- Frpc
- Nps 内网穿透

驱动增加：
- USB 2.0, USB 3.0
- Android 设备共享上网、USB上网卡上网
- 部分无线 Wireless 上网卡驱动

网络功能增加：
- 多WAN口负载均衡
- Turbo ACC 网络加速
